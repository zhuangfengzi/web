🛠️ 网页端连接问题解决指南

🔑 问题1: AppID配置问题

错误提示: "AppID配置为空" 或 "配置获取失败"

  1. 确保URL中包含AppID参数：
    https://your-domain.com/room.html?id=房间号&appId=你的AppID
  2. 在阿里云控制台获取正确的AppID：
    • 登录阿里云控制台
    • 进入"实时音视频通信RTC"产品页
    • 在"应用管理"中查看或创建应用
    • 复制AppID
  3. 如果没有AppID参数，页面会自动提示输入
🌐 问题2: 阿里云RTC SDK加载失败

错误提示: "阿里云RTC SDK加载失败，请检查网络连接"

  1. 检查网络连接是否正常
  2. 确认防火墙没有阻止外部JavaScript文件加载
  3. 尝试刷新页面重新加载SDK
  4. 检查浏览器控制台是否有其他网络错误
  5. 如果在企业网络环境，可能需要联系网络管理员
🎥 问题3: 摄像头/麦克风权限问题

错误提示: "请允许访问摄像头和麦克风权限"

  1. 在浏览器地址栏左侧找到摄像头/麦克风图标
  2. 点击图标，选择"允许"权限
  3. 如果没有弹出权限请求，刷新页面重试
  4. 检查浏览器设置中的摄像头/麦克风权限
  5. 确保设备的摄像头和麦克风硬件正常工作
🔗 问题4: 加入房间失败

错误提示: "加入房间失败" 或连接超时

  1. 检查房间ID是否正确
  2. 确认AppID配置正确
  3. 检查网络连接稳定性
  4. 尝试关闭VPN或代理
  5. 确认阿里云RTC服务正常运行
⚠️ 浏览器兼容性提醒

建议使用以下浏览器获得最佳体验：

  • Chrome 70+ (推荐)
  • Firefox 65+
  • Safari 12+
  • Edge 79+
